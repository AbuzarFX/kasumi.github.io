Commands of Kasumi
We provide you the best features in Kasumi that a music bot has to offer! Here are the commands of Kasumi.
Playback Commands

/p (song) - Plays the desired song.
/pause - Pauses the current song.
/resume - Resumes the paused song.
/stop (dc) - Stops/disconnects the music in the voice channel.
/skip - Skips to the next song.
/q (queue) - Returns the server queue.
/shuffle - Shuffles the current queue.
/np - Returns the NowPlaying song.
/ly - Get the lyrics of the current playing song.
/loop - Loops the current queue.
/volume (1-100%) - Changes the volume of the bot.

Utility Commands

/invite - Returns the Invite link of the bot.
/bot-info - Returns the information about the bot.
/uptime - Returns the uptime of the bot.
/ping - Returns the Latency of the bot.
/vote - Returns the Voting link for Kasumi on Top.gg. Thanks for voting ^_^
/bugreport (issue) - Used to report any bugs noticed in the bot. The developer will fix the bug (if it exists) shortly.
/yt - Used to search for a video on YouTube.
